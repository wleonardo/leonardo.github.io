1.apply中的使用

当我们的函数有多个变量的时候（特别是当我们不知道变量的数量的时候），有时候会通过将变量保存在数组中，并通过call来执行函数，有了展开运算符后则有了更好的方式（毕竟使用call需要手动指定this，有时候会不是很方便很准确）
